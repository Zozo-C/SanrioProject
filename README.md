# 🌸 小葵的三麗鷗夢想舞台

一個為小葵參加三麗鷗小小舞者比賽而設計的互動式應援網頁，結合徽章收集系統與打卡機制，讓支持者每天投票並獲得可愛的徽章獎勵！

## ✨ 功能特色

### 🎀 主題切換系統
- 5 種三麗鷗角色主題可選擇
- Melody、Cinnamoroll、Kuromi、布丁狗、Hello Kitty
- 每個主題都有獨特的配色與氛圍

### 🗳️ 每日投票打卡
- 點擊投票按鈕抽取當日徽章
- 防重複機制：每天只能領取一次
- 投票截止日期：2025年12月22日 23:59

### 🎁 徽章收集系統
- 30+ 種可愛徽章隨機抽取
- 超稀有隱藏徽章：向日葵 🌻（2% 中獎率）
- 中獎可兌換 45 元飲料一杯！
- 3 種展示模式：小卡、大卡、輪播
- 4 種徽章造型：預設、圓形泡泡、郵票風格、發光特效

### 💬 每日小語系統
- 根據打卡天數顯示不同的劇情語錄
- 開始、中間、結束三階段故事線
- 讓應援過程更有溫度與互動感

### 📷 分享應援卡
- 自訂祝福語
- 5 種動態卡片樣式（隨打卡天數變化）
- 一鍵下載圖片分享到 Instagram
- 顯示個人應援統計資料

### 📊 旅程進度追蹤
- 視覺化進度條顯示收集進度
- 統計已收集徽章數量
- 距離目標完成度一目了然

### ⭐ 背景特效
- 動態星星背景
- 隨徽章數量增加而變多
- 中大獎時觸發特殊慶祝效果

## 🚀 使用方式

### 線上瀏覽
直接訪問 GitHub Pages：
```
https://zozo-c.github.io/SanrioProject/
```

### 本地運行
1. 下載專案
```bash
git clone https://github.com/Zozo-C/SanrioProject.git
cd SanrioProject
```

2. 使用任何靜態伺服器開啟，例如：
```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx serve
```

3. 在瀏覽器開啟 `http://localhost:8000`

## 💾 資料儲存

所有資料都儲存在瀏覽器的 LocalStorage 中，包括：
- 個人暱稱
- 已選擇的主題
- 收集到的徽章
- 投票紀錄
- 個人設定（展示模式、徽章造型等）

⚠️ **注意**：使用無痕模式或清除瀏覽器資料會導致進度遺失！

## 🎨 技術細節

- **純前端實作**：HTML + CSS + JavaScript
- **無需後端**：使用 LocalStorage 儲存資料
- **響應式設計**：支援手機、平板、電腦
- **截圖功能**：使用 html2canvas 函式庫
- **字型**：Google Font - Zen Maru Gothic
- **色彩無障礙**：優化對比度確保可讀性
